# Ecommerce Sales SQL Analysis

Project Overview
This project uses SQL to analyze an ecommerce dataset containing customer orders, product details, and transaction data.  
The goal of the project is to answer key business questions related to sales performance, customer behavior, and revenue trends, helping businesses make data-driven decisions.


Questions:

1. What is the total revenue generated?  
2. What is the average order value?  
3. Which category generates the highest revenue?  
4. Which cities have the highest average order value?  
5. Which category generates the highest revenue?  
6. Which are the categories with the highest performance?  
7. What is the average order value for each customer?  
8. What is the total revenue generated each month?  
9. Which payment method is used the most by customers?  
10. Which payment method generates the highest revenue?  
11. What percentage of total revenue is contributed by each category?  



Results:

- Mumbai and Delhi emerged as the top-performing cities in terms of both order volume and revenue  
- UPI and Credit Cards were the most preferred payment methods, especially for high-value purchases, indicating strong digital payment adoption  
- Certain product categories such as kitchen appliances, small electronics, and some fashion items showed lower sales performance
